## About

This repository contains a **Kanban Board application**.  
It is a task management tool that helps users organize, prioritize, and track work efficiently using a drag-and-drop interface.  

**Key Features:**
- Create, edit, and delete tasks
- Drag and drop tasks across different columns (To Do, In Progress, Done)
- Assign tags, priorities, and deadlines to tasks
- Fully responsive and interactive UI for smooth task management

Built with **React**, **TypeScript**, and **Tailwind CSS**, this project focuses on **usability, performance, and modern design** for effective productivity.
